## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev
```

## Build & Deploy

```bash
# Build for production
npm run build

# Deploy to GitHub Pages
npm run deploy
```

## GitHub Pages Setup

1. Create a new repository on GitHub
2. Update the `base` path in `vite.config.js` to match your repo name (if not using user/organization pages)
3. Or set the `GITHUB_REPO_NAME` environment variable before building
4. Run `npm run deploy` to build and deploy
